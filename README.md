# Bike_Sales_Analysis
A complete Bike Sales data analysis project built using Microsoft Excel.
This project reads raw customer and sales data, cleans it, performs exploratory data analysis (EDA), and builds an interactive dashboard to visualize insights.
The project follows a real Excel tutorial covering data cleaning, pivot tables, visuals, and dashboard creation.

# Dataset Description
The dataset contains customer and sales information related to bike purchases, with the following fields:
-ID.
-Marital Status.
-Gender.
-Income.
-Children.
-Education.
-Occupation.
-Home Owner.
-Cars.
-Commute Distance.
-Region.
-Age.
-Purchased Bike (Yes / No).

# Project Goals
-Clean & preprocess the data.
-Handle duplicate rows and inconsistent formatting.
-Transform data into usable formats (e.g., gender labels, age groups).
-Perform exploratory analysis to understand patterns.
-Create PivotTables and charts.
-Build an Interactive Excel Dashboard.

# Tools & Techniques Used
-Microsoft Excel.
-Data Cleaning.
-Formulas & Functions (e.g., IF, COUNTIFS, SUMIFS).
-Pivot Tables.
-Pivot Charts & Visualization.
-Dashboard Design with Slicers.
